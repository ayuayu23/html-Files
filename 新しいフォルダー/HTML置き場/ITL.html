html css

Linux

基本となる知識が付くまでは教科書に沿って授業が進行していきます。また、使用する実機にLinuxがインストールされているわけではなくsshを用いてリモートコンピュータと通信して行います。

Java

Javaプログラミング能力認定試験の資格取得を目標にすることになります。最初にJavaとはどのようなものなのかと実機を使用しながらJavaの基本的なプログラミングの手順を学びJavaで使用するコマンドを覚えるために問題集などをしようしながら授業を進めることになります。
